# 第一阶段验收记录

## Step 1：ArkWeb 加载 H5 页面

目标：
验证 ArkTS Web 组件能加载 rawfile/h5/index.html。

当前不做：
- JavaScriptProxy
- runJavaScript
- Controller
- Dispatcher
- Register
- Biz / Imp

验收标准：
1. 应用启动后能看到 ASCF Runtime Lab 页面。
2. 页面显示 H5 内容。
3. DevEco 日志中能看到 onPageBegin / onPageEnd。
4. 不出现白屏或资源加载失败。

## Step 2：JavaScriptProxy 注入 ascfBridge

目标：
验证 H5 能通过 window.ascfBridge.send 调用 ArkTS 方法。

当前调用链路：
H5 Button
  ↓
window.ascfBridge.send(rawReq)
  ↓
JavaScriptProxy
  ↓
AscfBridgeObject.send(rawReq)
  ↓
return response string
  ↓
H5 页面展示 response

验收标准：
1. 页面能看到 “Step 2：调用 ascfBridge.send” 按钮。
2. 点击按钮后，DevEco 日志能看到 ArkTS 收到 rawReq。
3. H5 页面能展示 ArkTS 返回的 response。
4. response 中包含 requestId、code、message、data。
5. 当前阶段不使用 runJavaScript。
