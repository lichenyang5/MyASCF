视图层框架详细说明
基于vue3扩展的视图框架，为ASCF提供UI组件系统和渲染能力
目录结构
视图层
    src
        platforms
            fastapp
                fastapp.js 框架初始化入口
                component
                    view 容器组件
                    text 文本组件
                    ...
                    webView 网页容器
                    open-embedded-atomicservice 嵌入式元服务
                    index.js 组件统一导出
            page 页面管理
                index.js 页面生命周期
                page-mixin.js 页面混入
            style 样式管理
                style-parser.js 样式解析
                style-binding.js 样式绑定
            bridge 桥接
            mixins 混入
        shared 共享代码
        test 测试
    types 

fastapp.js 框架初始化
